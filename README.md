# PowerFit Welcome Website

This project is a **responsive gym landing page** created as part of the **AGNEYRA Front-End Development Internship Challenge**. The website is designed to welcome users to the PowerFit Gym and provide an engaging introduction to its services.

## 📌 Project Objective

The main objective of this project is to design a clean and responsive fitness website using basic front-end technologies while implementing layout design and interactive UI components.

## 🚀 Features

* Navigation bar with logo and menu links
* Hero section with heading and "Join Now" button
* Three-column service section to showcase gym services
* Modal popup that opens when the "Join Now" button is clicked
* Responsive layout for better user experience

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Modal popup functionality
